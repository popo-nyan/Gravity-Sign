# Sign HTTP 署名

Gravityでは、**sign**と呼ばれるHTTP署名を使用しています。  
この署名は通常、送信データの整合性を確保する為に使用されます。

# 注意
リクエストする際には、**t_secret**を消してください。  
[check_install.py](https://github.com/popo-nyan/Gravity-Sign/blob/master/sign.py)参照
